Project Description
효성 ITX 메타버스 사이트(관리자 페이지)

담당 업무
- Backend Rest API Server 개발
- 센터 관리,통계,모니터링,설정 메뉴 Frontend 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8 JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: MariaDB
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**리눅스 서버 환경 구축 경험과 이해**
이번 프로젝트를 진행하면서 리눅스 서버 환경을 직접 경험하며 다양한 명령어와 기본적인 서버 운영 방식에 대해 학습할 수 있었습니다. 사내에서 구축한 VMware 서버를 활용해 가상 서버를 추가로 생성하고, 이를 개발 서버로 구성하여 프로젝트에 적용하였습니다.
특히 처음으로 리눅스 서버에 개발 환경을 직접 구축해보며, 그동안 익숙하지 않았던 리눅스 명령어들을 실제 업무에 활용하는 경험을 하게 되었습니다. 이를 통해 서버 환경에 대한 이해도가 높아졌을 뿐만 아니라, 리눅스 기반 시스템 전반에 대한 흥미와 관심도 함께 생기게 되었습니다. 이번 경험을 계기로 서버와 인프라 영역에 대한 학습을 지속하며, 보다 안정적인 개발 환경을 구축할 수 있는 개발자로 성장하고자 합니다.
