Project Description
소프트폰 CTI Toobar Application 개발(리뉴얼 프로젝트)

담당 업무
- CTI 미들웨어 업체 교체로 소프트폰 API 변경 및 적용
- CTI 로그인, CTI 로그아웃, 전화 걸기, 전화 받기, 전화 끊기, 보류, 보류 해제, 협의, 호전환, 녹음 시작, 녹음 종료 기능 Frontend 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8, JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: MariaDB
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**유사 API 구조 변경 및 관리 경험**
이번 프로젝트를 통해 구조와 기능이 유사한 API들을 효율적으로 변경하고 관리하는 방법을 배울 수 있었습니다. 공통된 로직과 패턴을 파악하여 중복을 최소화하고, 변경 사항을 일관성 있게 적용하는 과정에서 유지보수성과 확장성을 고려한 개발의 중요성을 이해하게 되었습니다. 이 경험을 통해 향후 API 수정 및 확장 작업 시 보다 체계적으로 접근할 수 있는 역량을 갖추게 되었습니다.
