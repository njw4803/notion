Project Description
Nexus Solution 고도화 작업

담당 업무
- Java8을 Java17 버전으로 upgrade
- Spring Legacy System을 Spring boot로 변환
- Redis Session Clustering 설정 변경 
- 기존 기능 오류 분석 및 해결
- 공지사항 그룹,팀 발송 추가 기능 개발
- 공지사항 알림 기능 추가 기능 개발 - SSE (Server-Sent Event) 와 메시지 브로커로 Redis pub/sub 모델 활용

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 17, JavaScript
- 프레임워크: Spring Boot 3.2.2
- 서버: CentOS 7
- DB: MariaDB, Redis
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
  - Lombok
- 기타 협업 관리 도구
  - GIT

### 프로젝트를 통해 얻은 것

**부족한 지식
이번 프로젝트를 통해 Redis Session과 SSE와 같은 기술을 처음으로 실제 구현해보았습니다. 개념적으로는 알고 있었지만 직접 적용하는 과정에서 여러 시행착오를 겪었고, 그 경험을 바탕으로 학습을 통해 구현에 성공할 수 있었습니다. 이를 통해 꾸준히 공부해왔다고 생각했음에도 불구하고 아직 접해보지 못한 기술이 많다는 점을 깨달았으며, 현재의 부족한 부분을 명확히 인지하게 되었습니다. 앞으로도 새로운 기술에 대한 학습을 멈추지 않고, 실습과 경험을 통해 역량을 지속적으로 향상 시켜 나가고자 합니다.**
