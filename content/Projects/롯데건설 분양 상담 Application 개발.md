Project Description
롯데건설 분양 상담 Application 개발

담당 업무
- 상담업무 - 상담현황, Fax 전송현황, SMS전송현황, 분양정보 sms 전송현황 기능 Frontend 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8 JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: Microsoft SQL Server
- 기타 협업 관리 도구
  - Git Lab

### 프로젝트를 통해 얻은 것

**보안과 기간계 데이터 통신에 대한 이해**
이번 프로젝트를 통해 보안의 중요성과 기간계 시스템 간 데이터 통신에 대해 깊이 있게 배울 수 있었습니다. 프로젝트 진행 과정에서 외부 서버와 통신하여 데이터를 연동하는 작업을 처음으로 경험하였으며, 이를 통해 시스템 간 데이터 흐름과 연계 구조에 대한 이해를 넓힐 수 있었습니다.
또한 SI 프로젝트 파견 근무를 통해 실제 현업에서 요구되는 보안 이슈를 직접 경험하며, 단순한 기능 구현을 넘어 보안 정책과 규칙을 준수하는 개발의 중요성을 체감하였습니다. 이 경험을 통해 안정적인 서비스 제공을 위해서는 기술적 완성도뿐만 아니라 보안을 고려한 설계와 개발이 필수적이라는 점을 깨달았으며, 앞으로도 보안을 염두에 둔 개발 역량을 지속적으로 강화해 나가고자 합니다.
