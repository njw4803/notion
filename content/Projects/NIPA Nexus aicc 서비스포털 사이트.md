Project Description
NIPA Nexus aicc 서비스포털 사이트

담당 업무
- Backend Rest API Server 개발
- DB Procedure(프로시저) 작업
- 구독하기, 구독 후 서비스 신청 페이지(대시보드, 서비스 관리, 계약 관리, 결제관리, 계정관리) 기능 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 17, JavaScript
- 프레임워크: Spring Boot 3.1.2
- 서버: CentOS 7
- DB: MariaDB
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
  - Lombok
- 기타 협업 관리 도구
  - GIT

### 프로젝트를 통해 얻은 것

**개발자의 태도
이번 프로젝트에서는 15년 차 이상의 프리랜서 개발자 두 분과 협업을 진행하였습니다. 높은 연차의 개발자들이기에 많은 것을 배울 수 있을 것이라 기대했으나, 실제로는 기대와 다른 점이 많았습니다. 기본적인 개발 규칙을 준수하지 않거나 작업 속도 및 업무 집중도가 낮아 보이는 모습이 있었고, 전반적으로 책임감보다는 형식적으로 시간을 보내는 인상을 받았습니다. 그 결과 최종 결과물의 완성도가 기대에 미치지 못해, 대표님께 보고 후 해당 개발자들이 맡았던 부분을 다시 수정하는 과정을 거치게 되었습니다. 이 경험을 통해 연차나 경력보다 중요한 것은 개발자로서의 책임감과 태도라는 점을 다시 한번 깨달았으며, 앞으로 협업에 있어 올바른 개발 문화와 자세의 중요성을 더욱 중요하게 생각하게 되었습니다.**
