Project Description
티켓몬스터 상담,콜 현황 통계 화면 사이트 개발

담당 업무
- 계정관리, 큐등록관리, 응대/지원구분 관리, 예측호 관리 메뉴 기능 Backend, Frontend 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8 JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: Microsoft SQL Server
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**프로젝트 긴급 투입 경험과 아쉬움**
입사 후 약 한 달 간의 과제를 마친 뒤 투입된 프로젝트로, 이미 오픈 일정이 크게 지연된 상태에서 긴급하게 합류하게 되었습니다. 이로 인해 프로젝트 전반에 대한 충분한 이해 없이 개발을 시작해야 했던 점은 다소 아쉬움으로 남았습니다.
짧은 시간 안에 결과를 내야 하는 상황이었기에 주로 화면 개발을 중심으로 업무를 진행하였으며, 기본적인 CRUD 기능 구현이 주를 이루어 기술적인 난이도 자체는 크지 않았습니다. 다만 프로젝트의 배경과 목적을 충분히 파악하지 못한 상태에서 진행되다 보니, 보다 주도적으로 기여하지 못했다는 아쉬움이 있었습니다.
이번 경험을 통해 프로젝트 초반의 공유와 이해가 개발 효율과 완성도에 얼마나 중요한지 다시 한번 느끼게 되었으며, 향후에는 빠른 상황 파악과 적극적인 커뮤니케이션을 통해 더 높은 기여를 할 수 있도록 노력하고자 합니다.
