Project Description
자사 상담 Application Solution 고도화 및 리팩토링(Refactoring)

담당 업무
- Spring Legacy System 청산
- Spring Security 와 JWT적용 - 인증 , 인가 구현
- Log 기록 - AOP 공통 처리
- Controller Validation, Exception Handler 처리
- 채팅 서버 - 프로필 이미지 업로드 기능, Spring Websocket Stomp로 채팅 알림, 채팅 기능 구현 (Mongo DB 사용)
- Code Convention에 맞게 소스 수정
- SOLID원칙을 위반한 소스 수정
- DTO 분리 작업 - 각 도메인 기존 DTO를 inner class로 RequestDTO 와 ResponseDTO로 분리, 공통 변수는 추상클래스 CommonDTO에 inner class로 RequestCommonDTO 와 ResponseCommonDTO를 만들어 상속 받아 처리
- Mybatis 공통 SQL문 <include> 태그 활용하여 공통 처리

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 17, JavaScript
- 프레임워크: Spring Boot 3.1.2
- 서버: CentOS 7
- DB: MariaDB
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
  - Lombok
- 기타 협업 관리 도구
  - GIT

### 프로젝트를 통해 얻은 것

**자기계발의 중요성
자사 상담 Application Solution의 고도화 및 리팩토링 작업을 진행하면서, 더 나은 솔루션을 제공하기 위해 꾸준한 자기계발의 중요성을 깊이 체감하게 되었습니다. 해당 솔루션은 오래된 소스를 기반으로 하고 있어 개선이 필요한 부분이 많았으며, 유지보수와 확장성을 고려하지 않은 코드와 구식 기술로 구현된 영역들이 다수 존재했습니다. 이에 성능 개선과 장기적인 유지보수 및 확장성을 확보하기 위해 CTO이신 이사님과 충분한 논의 및 컨펌을 거친 후, 새로운 기술 도입과 소스 구조 개선을 단계적으로 진행하였습니다.**
최신 기술을 적극적으로 적용하고 싶은 욕심도 있었지만, 개인 개발이 아닌 팀 단위의 협업이었기에 무리한 도입보다는 동료 개발자들과의 충분한 소통을 통해 모두가 이해하고 활용할 수 있는 기술을 중심으로 방향을 설정하였습니다. 이 경험을 통해 자기계발은 개인의 역량 향상에 그치지 않고, 팀과 조직 전체의 기술 수준을 함께 끌어올리는 데 중요한 요소라는 점을 깨달았으며, 앞으로도 지속적인 학습과 공유를 통해 안정적이고 발전적인 개발 환경을 만들어가고자 합니다.
