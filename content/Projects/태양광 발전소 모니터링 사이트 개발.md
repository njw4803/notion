Project Description
태양광 발전소 모니터링 사이트 개발

담당 업무
- DB 및 알고리즘 설계
- Backend Rest API Server 개발
- 메뉴 권한 처리 - Spring Interceptor 처리
- 카카오 지도 API 활용한 메인 페이지 개발 (태양광 발전소 List 클릭 시 해당 지도 마크 표시 및 상세 정보 툴팁 기능, 마크 클릭 시 로그인 페이지 이동)
- 메모리 관리를 위한 DevExtreme API 공통 라이브러리 개발
- DevExtreme API 활용한 gird, chart 구현
- Excel 업로드, 다운로드 기능 구현
- 관리자 페이지, Xems, 수요공급 메뉴 Frontend 개발

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 17, JavaScript
- 프레임워크: Spring Boot 3
- 서버: CentOS 7
- DB: MariaDB
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
  - Lombok
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**DB 설계의 중요성**
이번 프로젝트에서는 PL 역할을 맡아 분석·설계 단계부터 직접 DB 설계를 진행하는 경험을 하였습니다. 화면 기획을 기반으로 필요한 컬럼을 정의하고, 테이블 분리와 관계도를 고려하며 구조를 설계하는 과정을 처음으로 주도적으로 수행하였습니다.
프로젝트 진행 중 화면 변경이나 추가 요구사항이 발생하면서 기존에 설계한 테이블 구조를 수정해야 하는 상황도 많아 어려움을 겪었지만, CTO 이사님의 조언을 바탕으로 유연하게 설계를 보완하며 문제를 해결해 나갔습니다. 이 경험을 통해 초기 DB 설계가 프로젝트 전반에 미치는 영향이 매우 크다는 점과 함께, 변화에 대응할 수 있는 확장성과 유연성을 고려한 설계의 중요성을 깊이 이해하게 되었습니다. 결과적으로 프로젝트를 성공적으로 마무리할 수 있었으며, 향후에는 더욱 안정적이고 체계적인 DB 설계를 할 수 있는 기반을 마련하는 계기가 되었습니다.
