Project Description
에너지 효율 시책 성과 평가 시스템 고도화 및 리팩토링(Refactoring)

담당 업무
- DB 마이그레이션, 개발 환경 구축 및 개발 시스템 분석
- 수요관리시책 에너지절감량 산정 지침을 반영하여 기 구축된 성과 평가 시스템 수정(입력 양식, 표출 화면 등) 구축
- 고효율기기 보급시책별 에너지절감성과 분석 기능 고도화
- 수요 관리 시책 간의 에너지절감량 중복성 확인을 위한 사업장 코드 관리 체계 정립 및 사업장 검색 기능 개발
- 비용효과 분석 기능 개발
- 시책 검증 메뉴 추가 개발
- 기존 소스 분석 후 excel 양식 업로드 기능 고도화 ( 속도 개선 - mybaties batch처리 )
- 첨부파일 업로드, 다운로드 기능 추가
- AS IS 오류 처리 및 권한 처리

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java8, JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: Oracle
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**엑셀 업로드 기능 고도화와 성능 최적화 경험**
이번 프로젝트에서는 기존 소스를 분석한 후, 엑셀 양식 업로드 기능을 고도화하는 작업을 수행하였습니다. 특히 대량 데이터 처리 속도를 개선하기 위해 MyBatis의 Batch 처리 기능을 적용하였으며, 이를 통해 업로드 성능을 크게 향상시킬 수 있었습니다.
이 경험을 통해 기존 시스템을 분석하고, 적절한 기술과 방법을 적용하여 성능을 최적화하는 과정의 중요성을 배웠습니다. 또한 대량 데이터 처리에서 효율적인 쿼리 설계와 배치 처리의 필요성을 깊이 이해하게 되었으며, 향후 성능 개선 작업에 자신감을 가지게 되었습니다.
