Project Description
유진투자증권 상담 콜 이력 통합/통계 사이트 개발

담당 업무
- 비밀번호 정규식(문자/숫자/특수문자 8자리 이상) 적용
- 비밀번호 ID와 동일 및 개인정보 사용 금지 설정 적용
- 분기별 1회 비밀번호 변경 경고, 비밀번호 미변경 시 계정 잠금 설정
- 비밀번호 5회 이상 실패 시 계정 잠금
- 오류 해결 지원(SQL문 수정)

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8, JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: MariaDB
- 라이브러리 및 기타 솔루션
  - Jquery 3.4.1
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

**강화된 비밀번호 정책 적용과 보안 이해**
이번 프로젝트에서는 다음과 같은 비밀번호 보안 정책을 적용하며 관련 지식을 습득할 수 있었습니다.
- 비밀번호 정규식 적용: 문자, 숫자, 특수문자를 포함한 8자리 이상 설정
- 비밀번호에 ID 및 개인정보 사용 금지
- 분기별 1회 비밀번호 변경 경고, 미변경 시 계정 잠금
- 비밀번호 5회 이상 실패 시 계정 잠금
이 과정을 통해 단순한 기능 구현을 넘어, 사용자 계정 보안을 강화하는 실질적 방법을 이해하게 되었습니다. 특히 정책 적용과 계정 보호가 서비스 안전성에 미치는 중요성을 체감했으며, 앞으로 개발 과정에서 보안을 설계 초기 단계부터 고려하는 습관의 필요성을 배우게 되었습니다.
