Project Description
국정원 견학,관광 상담 Application 개발

담당 업무
- Backend Rest API Server 개발
- 상담 저장, 통화 내역 현황, callback 기능, 블랙컨슈머 기능, 파일 (업로드,다운로드) 기능 Frontend 개발
- Spring Scheduler 어노테이션 @Scheduled 활용하여 매월 정해진 시간에 파일 테이블에 삭제 파일 데이터 삭제
- DB 백업 및 백업한 dump파일 storage 서버로 전송 -> stfp 전송 작업을 Linux Crontab으로 설정

### 프로젝트 상세 내용

**사용 기술 및 솔루션**
- 언어: Java 8 JavaScript
- 프레임워크: SpringFramework
- 서버: CentOS 7
- DB: MariaDB
- 기타 협업 관리 도구
  - SVN

### 프로젝트를 통해 얻은 것

이번 프로젝트를 통해 타 업체에서 제공하는 API를 활용한 데이터 연동 작업을 수행하며, 외부 시스템과의 연계 방식에 대해 배울 수 있었습니다. 또한 정해진 시간에 작업이 자동으로 수행되도록 스케줄링 처리를 구현하며, 서비스 운영 측면의 개발 경험을 쌓을 수 있었습니다.
아울러 DB 백업 작업과 리눅스 서버 환경에서 SFTP 전송을 자동화하기 위해 Crontab 설정을 적용하였으며, 이를 통해 서버 운영과 관련된 기본적인 자동화 프로세스를 이해하게 되었습니다. 이번 경험을 통해 단순한 기능 개발을 넘어, 안정적인 시스템 운영을 위한 백엔드 및 인프라 요소의 중요성을 인식하게 되었고, 향후에도 운영 환경을 고려한 개발 역량을 지속적으로 강화해 나가고자 합니다.
